### 📞 SELAI — Autonomous AI Sales Dialer
> *"AI handles the cold calls. Humans only join when there's a real buyer."*

[![Repo](https://img.shields.io/badge/GitHub-Selai-black?logo=github)](https://github.com/Sulayam/Selai)
![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?logo=typescript&logoColor=white)
![FastAPI](https://img.shields.io/badge/FastAPI-009688?logo=fastapi&logoColor=white)
![React](https://img.shields.io/badge/React-61DAFB?logo=react&logoColor=black)
![Gemini](https://img.shields.io/badge/Gemini-4285F4?logo=google&logoColor=white)

A real-time voice AI system where **autonomous agents handle outbound cold calls**, qualify buyer intent, and execute **warm transfers** to human salespeople only when a genuinely interested lead is detected — eliminating wasted sales rep time.

**Highlights:**
- AI agents autonomously dial leads and qualify intent end-to-end
- Gemini function calling detects real buyer signals mid-call
- Seamless **warm transfer** hands hot leads to human agents in real time
- Live React dashboard synced via WebSockets to every active call

**Stack:** TypeScript · Python · FastAPI · WebSockets · Gemini · STT/TTS · PSTN Routing · React
